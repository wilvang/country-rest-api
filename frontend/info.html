General Country Information

To retrieve information about a country, please add the ISO 3166-2 identifier of the country to the URL path.

Example: /countryinfo/v1/info/{ISO2}

You can also use the ?limit={0..} query parameter to limit the number of cities displayed (default is 10).

Example: /countryinfo/v1/info/{ISO2}?limit=5

Available paths:

  • Home
  • Population
  • Status
